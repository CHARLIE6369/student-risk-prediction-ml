🎓 Student Academic Risk Prediction using Machine Learning

This project predicts whether a student is at academic risk using ML algorithms.

🚀 Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib

📊 Problem Statement
Early identification of at-risk students helps institutions provide timely support.

🧠 ML Models Used
- Logistic Regression
- Decision Tree
- Random Forest

📁 Project Structure
See folders for data, notebooks, and source code.

▶️ How to Run
pip install -r requirements.txt
python src/train.py
